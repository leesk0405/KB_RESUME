Problems

  • 웹사이트의 부재
  • 사용법 안내 부족
  • 정보부재

국민은행 스타뱅킹앱에 대한 사이트의 부재로, 다양한 기능과 혜택을 이해하기 어렵습니다. 특히 계좌를 조회하고 이체하는지 등 실제적인 사용 방법에 대한 정보가 더욱 필요하며, 추가로 보안 기능과 사용자 편의성을 높이기 위한 팁과 부가적인 내용으로 앱을 보다 효과적으로 활용할 수 있도록 해야 합니다.

Solution

  • 웹사이트 제작
  • 사용법 안내
  • 필요한 정보 추가

국민은행은 스타뱅킹앱에 대한 전용 웹페이지를 제작하여, 앱의 핵심 기능, 편리한 서비스, 보안성 등을 자세히 설명하고 시각적으로 전달합니다.

이러한 해결책을 담은 웹사이트제작으로 국민은행 스타뱅킹앱에 대한 정보 부재 문제를 해결하고, 고객들에게 투명하고 효과적인 정보 전달을 가능하게 할 수 있습니다.

Component

Main Visual

국민은행 스타뱅킹에서 활용된 슬로건과 함께 브랜드 아이덴티티를 강조하는 시각적인 디자인으로 구성하였습니다. 더불어 배너에는 앱 버튼을 추가하여 향상된 접근성을 제공하여 사용자들이 쉽게 다운로드할 수 있도록 하였습니다.

Vision Component

국민은행 브랜드의 슬로건을 활용하여 국민은행의 전반적인 브랜드 아이덴티티를 명확하게 나타내었습니다.

Sign-in Guide Component

KB 스타뱅크의 간편회원가입을 통해 사용이 신속하고 간편한 프로세스를 이미지로 시각화하여 표현함으로써 이를 통해 쉽게 서비스를 이용할 수 있는 편리함을 강조하였습니다.

App guide Component

해당 페이지는 앱의 상세한 사용법을 이미지로 나타내어 사용자들에게 직관적이고 명확한 안내를 제공하였습니다.

Quick menu Component

KB 스타뱅크의 활용가이드로 편리하게 사용이 가능한 방법들을 한눈에 보기 편하도록 이미지를 활용하여 배치하였습니다.

FAQ Component

해당 페이지는 정보를 간결하게 구조화하고 사용자가 필요할 때 펼쳐보도록 하는 디자인 방식으로, 빠른 정보 접근과 업데이트 편의성을 제공하기위해 콜랩스 형태로 제작하였습니다.

application Component

KB스타뱅크 이외 국민은행의 다른 어플리케이션들을 이미지와 텍스트 형태로 swiper 라이브러리를 활용하여 슬라이드 형태로 배열하였습니다. 이를 통해 사용자들은 직관적으로 각 앱에 손쉽게 접근하여 편리하게 이용할 수 있도록 하였습니다.

News Component

해당 페이지는 제목과 함께 날짜와 슬라이드 형태로 배열되어 사용자들이 빠르게 최신 소식에 접근하고 손쉽게 이용할 수 있도록 디자인하였습니다.

UI Component

애플리케이션의 핵심적인 조작화면을 볼드한 타이포그래피와 함께 비주얼 요소로 활용하였습니다.

Footer Component

국민은행 footer를 참고하여 디자인을 새롭게 제작하였습니다. 상단에는 휴대폰 이미지와 다운로드 버튼을 추가하여 더욱 향상된 접근성을 제공하였습니다. 또한, 스크롤 시 배경 색이 변하며, 이는 언제 어디서든 편리하게 사용 가능하다는 의미를 인터랙션을 통해 강조하였습니다.
